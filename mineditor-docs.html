WYSIWYG HTML EDITTOR

USE MINEDITOR



Step 1: Include the MINEDITOR



Include this line of code in the <head> of your HTML page:



    <link rel="stylesheet"  href="https://maxcdn.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css">

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>


Include this line of code under the <body> of your HTML page:



    <script type="text/javascript" src=examples/js/keetool-editor.js></script>

Step 2: Initialize MINEDITOR as part of a web form



With the script included, initialize MINEDITOR on any element (or elements) in your web page.


In this example, let's create <div id = "mineditor" > with a MINEDITOR instance by passing the load and init('mini-editor') in Function window.addEventListener()
